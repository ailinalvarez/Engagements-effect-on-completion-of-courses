# EDA: Impacto del compromiso (Engagement) en la finalización de cursos

Este proyecto de Análisis Exploratorio de Datos (EDA) investiga la relación entre el nivel de compromiso de los estudiantes y su probabilidad de completar con éxito un curso en línea. 

## 📝 Introducción
El objetivo es definir qué significa estar "comprometido" en un entorno educativo digital y medir cómo estas métricas de comportamiento influyen en el logro de los objetivos académicos.

## 💡 Hipótesis
> "Un alto nivel de compromiso resulta en una mejora significativa y medible en las probabilidades de completar un curso."

## 📊 Resumen del Análisis
El estudio utiliza datos de comportamiento para identificar patrones entre los estudiantes. Algunos puntos clave abordados son:
- Preparación y limpieza de datos provenientes de Kaggle.
- Definición de métricas de compromiso (Engagement).
- Comparación de tasas de finalización entre estudiantes promedio y estudiantes altamente comprometidos.

## 🚀 Conclusiones Principales
A través del análisis, se determinó que:
- Un estudiante elegido al azar tiene aproximadamente un **39.64%** de probabilidad de completar el curso.
- Sin embargo, para un estudiante clasificado como **"altamente comprometido"**, la probabilidad de éxito aumenta a más del doble, alcanzando un **98.33%**.
- Aunque los puntos de datos individuales pueden no ser predictores perfectos por sí solos, su combinación permite identificar claramente a los estudiantes con mayores posibilidades de éxito.

## 🛠️ Tecnologías utilizadas
- Python 3
- Jupyter Notebooks
- Librerías de análisis de datos (Pandas, Matplotlib/Seaborn)
